This project is a Streamlit-based web application that recommends career paths based on user's skills, interests, education and experiences.

It utilizes a Naive Bayes machine learning model with word embeddings and cosine similarity to deliver productive and personalized recommendations based on the input parameters given by users. The system combines classification and semantic matching to enhance the accruacy and relevance of the suggested careers.

The main goal of this project was to offer assistance in career-making for the benefit of its users by providing data-driven insights on a user friendly interface.
